Comparativa de Lenguajes

Lenguaje Tipado Paradigma Compilación/Ejecución Usos más comunes
Python Dinámico Procedural, OO, Funcional Interpretado (CPython, PyPy) Web, Ciencia de datos, IA, Scripts
Java Estático Orientado a objetos Compila a bytecode → JVM Empresarial, Android, Big Data
C++ Estático Procedimental, OO, Genérico Compilado nativo Juegos, Sistemas, Alta perf.
C# Estático Orientado a objetos, Funcional Compila a MSIL → CLR Desktop, Web, Juegos (Unity)

Python

Python destaca por su sintaxis clara, tipado dinámico y gran ecosistema. Cada indentación define bloques, lo cual refuerza la legibilidad.

  • Ventajas: rápido prototipado, gran comunidad, bibliotecas de IA y datos.
  • Desventajas: rendimiento menor que C/C++, gestión de hilos limitada (GIL).

Ejemplo – List Comprehension y Función Recursiva:

# Filtrar números pares y calcular factorial
pares = [x for x in range(10) if x % 2 == 0]

def factorial(n):
    return 1 if n == 0 else n * factorial(n-1)

print("Pares:", pares)
print("5! =", factorial(5))

Java

Java usa tipado estático y corre en cualquier plataforma con una JVM. Su filosofía “Write Once, Run Anywhere” facilita la portabilidad.

  • Ventajas: robustez, recolector de basura, ecosistema empresarial.
  • Desventajas: sintaxis verbosa, consumo de memoria.

Ejemplo – Clase y Método Estático:

public class Calculadora {
    public static int suma(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        System.out.println("2 + 3 = " + suma(2, 3));
    }
}

oracle.com/java

C++

C++ ofrece control de bajo nivel con características de alto nivel: RAII, templates y sobrecarga de operadores.

  • Ventajas: máxima velocidad, uso en sistemas críticos.
  • Desventajas: gestión manual de memoria, complejidad.

Ejemplo – Plantilla Genérica y Punteros:

#include 
using namespace std;

template
T maximo(T a, T b) {
    return (a > b) ? a : b;
}

int main() {
    int *p = new int(42);
    cout << "Máximo(5,7) = " << maximo(5, 7) << endl;
    cout << "Valor apuntado: " << *p << endl;
    delete p;
    return 0;
}

C#

C# se integra con .NET, soporta LINQ, async/await y una sintaxis moderna inspirada en C++/Java.

  • Ventajas: desarrollo rápido, gran biblioteca estándar.
  • Desventajas: menor portabilidad fuera de .NET.

Ejemplo – LINQ y Async:

using System;
using System.Linq;
using System.Threading.Tasks;

class Programa {
    static async Task Main() {
        var nums = new[] {1,2,3,4,5};
        var pares = nums.Where(n => n % 2 == 0);
        Console.WriteLine("Pares: " + string.Join(",", pares));
        await Task.Delay(500);
        Console.WriteLine("¡Tarea asíncrona completada!");
    }
}
